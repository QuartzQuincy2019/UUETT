## 下载和使用方法
1. 点击页面右侧的**Releases**中的**Latest**，进入最新发布界面。
2. 在**Assets**下拉菜单中选择**Source Code (Zip)**，即可进行下载。
3. 下载解压后，用浏览器（**建议使用Microsoft Edge**）打开**UU.html**，进入使用。
4. 您的按键已被文件`database.js`中的按键映射绑定，无需准备任何输入法。
